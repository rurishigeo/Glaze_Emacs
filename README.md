## Overview

Glaze Emacs is a thoughtfully designed configuration aimed at enhancing your coding experience with a clean setup and a touch of sophistication. It integrates essential packages, visually pleasing themes, and meticulous settings to provide you with an efficient and aesthetically pleasing environment.

## Features

- **Modular Structure:** An organized configuration with modular files for easy management.
- **Simplified Configuration:** Utilizes straightforward packages to enhance functionality and productivity.
- **Elegant Themes:** Stylish and visually pleasing themes create a comfortable coding atmosphere.
- **Startup Optimization:** Fine-tuned settings to ensure a swift and efficient Emacs startup.

## Installation

1. Clone the repository to `~/.emacs.d`.
2. Start Emacs, and the configuration will be automatically loaded.
3. Run `M-x company-tabnine-install-binary` to use TabNine for auto-completion.
4. Run `M-x treesit-install-language-grammar` to generate the syntax dynamic library (make sure git and gcc are installed).

##Fonts

Install the following fonts or modify the relevant font information in lisp/init-theme.el.

- **Mac:** Monaco
- **Linux:** Hack Nerd Font
- **Windows:** Microsoft YaHei Mono

## Customization

Feel free to customize the configuration to suit your preferences. Explore the modular files in the `lisp` folder for specific settings.

## Usage

1. Open Emacs.
2. Enjoy the customized Glaze Emacs environment for a delightful coding experience.
